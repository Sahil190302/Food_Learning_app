# Daawat-e-Ishq – Recipe & Cooking Application

Daawat-e-Ishq is a modern Flutter-based recipe application designed to help users discover, explore, and manage recipes from multiple cuisines. The application provides a visually rich interface with categorized recipes, search functionality, and the ability for users to add their own recipes.

The project focuses on delivering an intuitive cooking experience through a clean UI, smooth animations, and structured content presentation.

---

# Application Overview

The application acts as a digital recipe collection platform where users can explore different cuisines, search recipes, and organize cooking ideas in a single place.

The app provides:

• Categorized recipe browsing  
• Modern animated user interface  
• Recipe discovery experience  
• Custom recipe creation  
• Organized navigation and content layout  

The application is built using Flutter and follows a modular screen-based architecture.

# Key Features

## 1. Modern Dashboard UI
The dashboard is the main landing screen that provides users with quick access to all application features.

Features include:

• Animated gradient background  
• Greeting section based on time of day  
• Search functionality  
• Recipe category grid  
• Featured recipe section  
• Floating action button for adding recipes  
• Bottom navigation bar  

---

## 2. Recipe Categories

Recipes are organized into different categories to simplify navigation and discovery.

Current categories include:

• Indian Veg  
• Continental  
• Healthy  
• Street Food  
• Desserts  
• Beverages  

Each category navigates to a dedicated screen displaying recipes for that cuisine.

---

## 3. Recipe Discovery

Users can browse through recipes and explore dishes from different food cultures.

Features include:

• Category-based navigation  
• Featured dishes section  
• Trending recipes design layout  
• Cooking duration indicators  
• Ratings display  

---

## 4. Add Recipe Feature

The application allows users to add new recipes directly from the dashboard.

When the floating action button (+) is tapped:

A separate screen opens where users can:

• Select a recipe category  
• Enter recipe title  
• Add ingredients  
• Add instructions or cooking steps  

This makes the application extensible and allows user-generated content.

---

## 5. Search Functionality

The dashboard includes a search bar that allows users to search for:

• Recipes  
• Ingredients  
• Food items  

This improves usability when the number of recipes grows.

---

## 6. Drawer Navigation

A navigation drawer provides access to additional application pages:

• Privacy Policy  
• Terms and Conditions  
• Return and Refund Policy  
• Logout  

Each item opens a separate screen containing detailed information.

---

## 7. Animated User Experience

The application uses multiple animation techniques to enhance visual experience.

Animations include:

• Fade transitions  
• Slide transitions  
• Gradient motion backgrounds  
• Animated category cards  

These animations provide a premium feel to the interface.
